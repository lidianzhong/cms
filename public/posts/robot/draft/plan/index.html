TimeSpec stamp();
uint32_t errorCode(); //错误代码

/*
运动模式
0. idle, default stand
1. balanceStand
2. pose
3. locomotion

6. jointLock
7. damping
8. recoveryStand
9. reserve
10. sit

消息案例：

Parameter: [ {
    "t_from_start": 0.0, "vx":0.38878634572029114, "vy":0.0, "vyaw":0.0, "x":0.3143965005874634, "y":0.0, "yaw":0.0
}

}

.......

]

消息含义解释：

  • "t_from_start": 从路径开始到达此点所需的时间。
  • "vx": 在此点的 x 方向上的期望速度。
  • "vy": 在此点的 y 方向上的期望速度。
  • "vyaw": 在此点的期望偏航速度。
  • "x": 此点的 x 坐标。
  • "y": 此点的 y 坐标。
  • "yaw": 此点的偏航角。
